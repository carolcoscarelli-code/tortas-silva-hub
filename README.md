# 🥧 Tortas dos Silva — Hub de Vendas

Sistema de gestão completo da **Tortas dos Silva** (Edição 2026).

🌐 **Acesse em:** https://carolcoscarelli-code.github.io/tortas-silva-hub/

## ✨ Funcionalidades

- 📊 Dashboard com KPIs e gráficos de vendas
- 💳 PDV (ponto de venda) com 2 canais (Cliente Final / Revenda)
- 📅 Encomendas com Kanban de produção
- 🍳 Planejamento de Produção com lista de compras
- 👩‍🍳 Fichas Técnicas (12 tortas, 9 empadões, 5 quiches, 9 receitas base)
- 🏷️ Precificação multi-canal (Loja, iFood, Revendas) com rateio de custos fixos
- 🥣 Cadastro de ingredientes com sincronização automática
- 🟢 Entradas (PDV + Encomendas + lançamentos manuais)
- 💸 Despesas reais com 10 categorias
- 📊 DRE (Demonstração do Resultado) com dados reais
- ☁️ Sincronização em tempo real entre dispositivos via Firebase

## 📱 Mobile-first

Bottom nav com 5 atalhos rápidos. Tabelas viram cards no celular. Modais full-screen.

## 🔐 Sincronização

Sistema usa Firebase Realtime Database. Código de sync embutido — nada a configurar.
