☁️ Setup da Nuvem

Sincronizar entre seu celular, computador e o Felipe — passo a passo

⏱️ Tempo total: ~5 minutos. Você só precisa fazer isso UMA VEZ. Depois, o Felipe e a mãe dele só vão precisar do código de sincronização.

📋 Checklist geral

  • Criar projeto no Firebase (gratuito)
  • Ativar Realtime Database
  • Liberar acesso (regras públicas)
  • Copiar as credenciais
  • Colar no sistema + escolher código de sincronização
  • Compartilhar URL + código com Felipe e a mãe

1️⃣ Criar projeto Firebase

1 Acesse o Firebase Console

Entre com sua conta Google (a mesma do gmail).

🔗 Abrir Firebase Console
2 Clique em "Criar um projeto" (ou "Add project")

Nome sugerido: tortas-silva-hub (qualquer nome funciona).

Próximas telas: pode desabilitar Google Analytics (não precisa) e clicar em "Criar projeto".

2️⃣ Ativar Realtime Database

3 No menu lateral, clique em "Build" → "Realtime Database"

(NÃO confunda com "Firestore Database" — use Realtime Database mesmo.)

4 Clique em "Criar banco de dados"
  • Localização: us-central1 (padrão, OK)
  • Modo: "Iniciar no modo de teste"
⚠️ O modo "teste" expira em 30 dias. Vamos configurar regras permanentes no próximo passo.
5 Configurar as regras (importante!)

Na aba "Regras" (do Realtime Database), apague tudo e cole:

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

Clique em "Publicar".

💡 Por que isso é seguro? A proteção fica no código de sincronização secreto que você vai escolher. Quem não souber o código, não acessa nada (nem sabe que existe). É como uma porta sem cadeado em uma rua escondida — quem não sabe o endereço, não chega.

3️⃣ Pegar as credenciais

6 Adicionar um "App Web"

No menu lateral, clique no ícone de engrenagem ⚙️ → "Configurações do projeto".

Role até o final, na seção "Seus apps", clique no ícone de web </>.

7 Registrar o app
  • Apelido: Tortas do Silva Web
  • NÃO marque "Firebase Hosting"
  • Clique em "Registrar app"
8 Copiar o objeto firebaseConfig

Vai aparecer um código tipo isso:

const firebaseConfig = {
  apiKey: "AIzaSyD...",
  authDomain: "tortas-silva-hub.firebaseapp.com",
  databaseURL: "https://tortas-silva-hub-default-rtdb.firebaseio.com",
  projectId: "tortas-silva-hub",
  storageBucket: "tortas-silva-hub.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123:web:abc..."
};

Copie tudo do { até o } (inclusive as chaves).

⚠️ Se não aparecer o campo databaseURL, volte na seção do Realtime Database e copie a URL que aparece em cima (parecida com https://seu-projeto-default-rtdb.firebaseio.com) e adicione manualmente no objeto.

4️⃣ Colar no sistema

9 Abra o sistema (index.html)

No canto superior direito, clique no indicador ⚪ Local.

(Se não aparecer, vai em Configurações → ☁️ Sincronização na Nuvem.)

10 Cole a configuração

No campo "Configuração do Firebase", cole o objeto {...} que você copiou.

11 Defina o código de sincronização

Sugestão: tortas-silva-2026 ou algo memorável só pra vocês.

Esse código é a "senha" pra entrar no workspace. Quem souber, acessa.

12 Clique em "💾 Conectar"

Pronto! O indicador vai virar ☁️ Sincronizado. Tudo que você cadastrar a partir daqui é enviado pra nuvem em segundos.

5️⃣ Compartilhar com Felipe e a mãe

13 Hospedar online (1 vez só)

Pra Felipe e a mãe acessarem do celular deles, o sistema precisa estar numa URL pública. Você pode usar:

  • GitHub Pages (você já usou pra Chocake): suba o index.html pra um repositório novo, ative Pages
  • Netlify Drop: arrasta o arquivo em app.netlify.com/drop — sai uma URL na hora

Funciona qualquer um dos dois — escolhe o que preferir.

14 Mandar pro Felipe e pra mãe

Mande no WhatsApp:

  • 📱 URL do sistema
  • 🔑 Configuração Firebase (o objeto {...})
  • 🔐 Código de sincronização (ex: tortas-silva-2026)

Eles abrem a URL, clicam no indicador ⚪ Local, colam a config e o código → pronto. Sincronizados.

✅ Como saber se está funcionando

No canto superior direito da tela aparece ☁️ Sincronizado em verde
Cadastra uma despesa no celular → abre no PC e ela aparece em ~2 segundos
Felipe lança uma encomenda → aparece pra você instantaneamente

🆘 Se der problema

Indicador fica ⚠️ Erro
  • Verifique se copiou o databaseURL certo
  • Vai no Firebase Console → Realtime Database → Regras → confirme que tá ".read": true, ".write": true
Mudou de código por engano

Vai em Configurações → Sincronização na Nuvem → Desconectar, e cadastra de novo com o código certo.
